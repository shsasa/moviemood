### **_Description_**


MovieMood is a web app that helps you keep track of the movies you want to watch. You can create custom lists based on your mood or theme, mark movies as watched or unwatched, and write reviews. It‚Äôs your personal space to organize and enjoy your movie-watching experience.

#### **_Find your next favorite film, plan the perfect movie night, and bring your mood to the screen ‚Äî all with MovieMood! üé¨üçø_**

---

### **_Technologies Used_**

- Node.js
- Express.js
- MongoDB + Mongoose
- JavaScript / HTML / CSS
- EJS
- Authentication & Session Middleware

## **_Special Features_**

1. User authentication and profile management
2. Create and manage personal movie entries
3. Custom lists by mood/theme
4. Watch/unWatch toggle
5. Add, edit, and delete personal reviews
